td:nth-of-type(1) : 상위 요소를 표시하는 형제 요소를 처음으로 만든 td요소를 선택 각 행의 첫 번째 셀을 선택 CSS 의사 클래스 선택자는 상태 또는 다른 요소와의 관계를 기반으로 요소를 선택하고 스타일을 지정하는 데 사용 < 자주사용하는 CSS 유사 클래스 선택자 > :hover - 사용자가 요소 위로 마우스를 가져가면 요소를 선택합니다. :active - 요소를 선택하는 동안 요소를 선택합니다. :focus - 현재 포커스가 있는 요소를 선택합니다(예: 입력 필드가 선택된 경우). :visited - 선택 : 첫 번째 자식 :last-child - 부모의 마지막 자식 요소를 선택합니다. :nth-child(n) - 부모의 n번째 자식 요소를 선택합니다. :n번째 :not(selector) - 주어진 선택자와 일치하지 않는 요소를 선택합니다. :empty - 자식이나 텍스트 콘텐츠가 없는 요소를 선택합니다
